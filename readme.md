# 🛒 Smart Product Comparator

A modern and interactive **Product Comparison UI** built using HTML, CSS, and Vanilla JavaScript.

This project allows users to compare multiple products side-by-side with detailed specifications in a clean SaaS-style interface.

## 🎯 Project Purpose

This project was created to practice and demonstrate:

- UI design for SaaS-style dashboards
- Product comparison logic
- DOM manipulation with JavaScript
- Responsive layout building
- Clean component-based frontend structure

---

## ✨ Features

- 🛒 Product comparison cards
- 📱 Mobile-friendly responsive design
- ⭐ Highlight best product selection
- 🖼️ Product images (stable placeholders)
- 🎨 Modern glassmorphism UI
- ✨ Hover animations and transitions
- 🧠 Simple interactive selection system

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Glassmorphism)
- JavaScript (Vanilla JS)
- Responsive Web Design
